Version: Albedo Toolbox 1.0

*********************************************************************
* About
*********************************************************************

The Albedo Toolbox is a Matlab/Simulink toolbox which provides 
functions for calculating the Earth albedo illuminating a spacecraft.
The Earth albedo model is based on TOMS reflectivity data.

*********************************************************************
* Installation
*********************************************************************

Unzip all files and specify the location of the directory containing
this file in the Matlab PathTool. TOMS reflectivity data is available
from the TOMS website as txt files or from the toolbox author website
as mat files. Place mat files in refl_data sub-folder with
sub-folders for each year, which will enable automated access for the
toolbox. Otherwise the path to the files must be specified in the 
Matlab PathTool.

*********************************************************************
* Compatibility
*********************************************************************

Albedo Toolbox 1.0 has been tested in Windows and Solaris.

Version details:
Matlab 7.1.0.183 (R14) Service Pack 3
Simulink 6.3
